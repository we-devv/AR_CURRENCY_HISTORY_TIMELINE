> Visualize Forex market trends in Augmented Reality on a vertical plane.

**ARCurrencyTimeline** is a standalone Android application built with Unity that generates a linear graph of Forex (Foreign Exchange) rates. Using the Alpha Vantage API, it fetches real-time data and visualizes currency timelines directly on your wall or any vertical surface.

## ğŸ“¥ Download & Installation

The **`ARCurrencyTimeline.apk`** file is available for direct download via the GitHub **Releases** page. This is the simplest way to get the app on your Android device.

1.  **Go to Releases:** Click on the **[Releases](../../releases)** tab (or look in the right sidebar) of this repository.
2.  **Download the APK:** Download the latest version of `ARCurrencyTimeline.apk`.
3.  **Enable Installation:** On your Android phone, you may need to go to **Settings > Security** and enable **"Install from Unknown Sources"** to allow the installation.
4.  **Install & Run:** Tap the downloaded file to install the app.
5.  **Usage:** Point your camera at a flat, textured vertical surface (like a wall) to detect the plane and place the interactive AR graph.

## ğŸŒŸ Features

* **AR Visualization:** Projects a 3D line graph of currency data onto the real world using Augmented Reality.
* **Vertical Plane Detection:** Optimized for walls and vertical surfaces using AR Foundation.
* **Data Source:** Uses the **Alpha Vantage API** for near real-time Forex market data.
